**VARIABLES**
- Before ES6(2015), var was the only way to create variables in js.
- javascript provides 3 ways to declare variables
    var
    let (chances of erros)
    const (by default we must use const)

# var
var a = 10
var a ==> declaration
a = 10 ==> initialization
redeclaration and reinitialization is possible (disadvantage)
hoisting is possible with var
var is a functional scope
var is not a block scope

# let
let is a block scope and a functional scope but basically it is a *block scope*
redectlaration is not possible but reinitialization is possible

# const
redectlaration and reinittialization is not possible
const is a block scope and functional scope but basically it is a *block scope*
practically const is prefered
